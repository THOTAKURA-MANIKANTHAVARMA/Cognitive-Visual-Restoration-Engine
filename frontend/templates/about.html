🧠 Powered by Deep Learning

AI Models & Architectures

Two state-of-the-art PyTorch CNN architectures by Zhang et al. — trained on over 1 million ImageNet images — transform grayscale images into vibrant, photorealistic color using LAB color space prediction.

Automatic Colorization CNN · PyTorch
🧠

ECCV 16 — Zhang et al. (2016)

A fully automatic deep CNN that takes a grayscale L channel input and predicts the ab chrominance channels in the LAB color space. The ab output is quantized into 313 color bins and treated as a classification problem with a class-rebalancing softmax loss, producing plausible, natural-looking colors. Weights are loaded from colorization_release_v2-9b330a0b.pth via PyTorch.

LAB Color Space 313 ab Bins PyTorch Softmax Loss ImageNet Pretrained
Primary · High Quality CNN · PyTorch
🎨

SIGGRAPH 17 — Zhang et al. (2017)

An enhanced U-Net style deep CNN with skip connections that supports user-guided colorization. Takes the grayscale L channel along with optional hint color inputs and outputs ab chrominance channels via a regression head. Produces more vibrant, photorealistic colors and is the primary inference model used in this application. Weights are loaded from siggraph17-df00044c.pth via PyTorch.

LAB Color Space U-Net + Skip Connections PyTorch Regression Output User-Guided

🔬 Methodology

01

Data Collection

Curated a dataset of color images spanning various categories including landscapes, portraits, and architecture. Images were converted to grayscale to create training pairs for supervised learning.

02

Model Architecture

Two PyTorch CNN models are used: ECCV16 (Zhang et al., 2016) — which treats colorization as a 313-bin ab classification problem — and SIGGRAPH17 (Zhang et al., 2017) — a U-Net style network with skip connections that regresses ab values directly, producing more vibrant results. SIGGRAPH17 is used for all inference in this app.

03

Model Loading

Pre-trained weights are loaded via PyTorch from local .pth files (colorization_release_v2-9b330a0b.pth for ECCV16 and siggraph17-df00044c.pth for SIGGRAPH17). Images are preprocessed into the LAB color space using skimage, and results are post-processed back to RGB with matplotlib for saving.

04

Evaluation Metrics

Performance evaluated using PSNR (Peak Signal-to-Noise Ratio), SSIM (Structural Similarity Index), and perceptual loss metrics. Achieved 28.5 dB PSNR and 0.92 SSIM on validation set.
